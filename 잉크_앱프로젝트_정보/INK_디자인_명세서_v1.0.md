# INK 디자인 명세서 v1.0 (최종본)

---

## 0) 디자인 목표

- **손글씨(캔버스) 중심**: 채팅방 기본 화면은 텍스트 타임라인이 아니라 "실시간 캔버스"
- **iOS/iPadOS 26 감성**: 심플, 시트/레이어 기반, 컨텍스트 메뉴 일관
- **반응형**: 가로/세로 + iPhone/iPad/PC 대응
- **우선순위**: 실시간 체감(즉시 반응) > 보안/권한(명확한 상태)

---

## 1) 정보구조(IA) & 네비게이션

### 1.1 하단 탭 (동결)

| 순서 | 탭 |
|------|-----|
| 1 | 친구 |
| 2 | 채팅 |
| 3 | 모아보기(태그함) |
| 4 | 설정 |

- **"게시판" 기능 없음(완전 제거)**. "모아보기(태그함)"으로 대체

### 1.2 네비 패턴

| 디바이스 | 패턴 |
|----------|------|
| **iPhone** | 탭 기반 + 화면 전환(push), 시트는 바텀시트 중심 |
| **iPad** | 디스코드형 3패널(좌 네비/중 목록/우 콘텐츠) + 시트/오버레이 |

---

## 2) 반응형 레이아웃 규칙

### 2.1 iPad 가로 (기본)

- 좌측 패널(네비) 고정
- 중앙: 채팅방 목록(퍼즐 카드)
- 우측: 선택된 화면(캔버스/상세)

### 2.2 iPad 세로 / 작은 폭

- 좌측 패널 접힘 → 버튼/스와이프 오버레이
- 사이트 시트는 부분 높이 시트로 동작

### 2.3 iPhone

- 목록 1열 → 방 진입 시 캔버스 전면
- 메뉴/설정/필터는 바텀시트로 제공

---

## 3) 로그인 이후 진입 구조

### 3.1 iPad

- 로그인 완료 → 채팅(메인)
- 우측 상단 메뉴바 버튼 → 사이트 시트(Site Sheet) 토글
- 시트에서 항목 선택 후 **"들어가기"**로 확정 진입

### 3.2 iPhone

- 로그인 완료 → 채팅(메인)
- 사이트 시트는 필요 시 호출

---

## 4) 채팅 메인(방 목록) — 퍼즐 카드 UI

### 4.1 카드 구성

- 방 이름 / 마지막 활동 시간 / 미확인 배지
- **마지막 손글씨 미리보기(퍼즐 타일)**
  - 마지막 손글씨를 타일 조각처럼 분할 배치해 "퍼즐 느낌" 연출
- (권장) 마지막 이벤트 타입 아이콘: 손글씨/사진/영상/PDF

### 4.2 상호작용

| 동작 | 결과 |
|------|------|
| **탭** | 해당 화이트보드 대화방(캔버스) 진입 |
| **롱프레스** | 방 컨텍스트 메뉴(알림/고정/나가기/초대 등) |

---

## 5) 화이트보드 대화방(캔버스) UI

### 5.1 상단 바 (공통)

- 방 이름/멤버수, 검색, 자료(공유 파일/미디어), 더보기(⋯)

### 5.2 상단 중앙: 펜 툴바

- 펜(슬롯), 지우개, 선택(라쏘/사각), 색 슬롯, 굵기 슬롯, Undo/Redo, 도형

### 5.3 상단 우측: 방 기능 버튼

- 내보내기 / 권한·설정 / 초대
- (선택) 링크 공유

### 5.4 좌측 패널 (iPad 디스코드형)

- 방 이동(최근/고정)
- 기능 숏컷: 도형/사진/영상/문서 삽입

---

## 6) 캔버스 입력/실시간 표현 규칙

### 6.1 입력 정책 (패드)

| 입력 | 동작 |
|------|------|
| **손가락** | 이동/선택/스크롤 |
| **펜** | 글쓰기/그리기만 |

### 6.2 고스트 라이팅

- 누군가 쓰는 중이면 **반투명(ghost)** 표시
- 입력이 멈추면 N초 후 **100% 확정**

### 6.3 작성자 표시 (오브젝트 인스펙트)

- 손글씨/사진/영상/PDF/텍스트/도형 위에서 작성자 확인 가능

| 디바이스 | 동작 |
|----------|------|
| **iPad (펜 호버)** | 커서가 오브젝트 위를 지나가면 작성자 툴팁 표시 |
| **iPhone** | 오브젝트 짧게 누르면 작성자 툴팁 표시 |

- 롱프레스는 편집 메뉴(Dim + 팝업)로 유지

---

## 7) 자동 색상 구분 (AUTO)

- 펜 스타일을 따로 지정하지 않은 기본 사용자는
- **사용자별 자동 색상(User Color)**로 스트로크/텍스트가 구분됨
- 사용자가 직접 색을 선택하면 그 설정이 우선
- (권장) 툴바에 AUTO 상태 표시 + 작성자 툴팁에도 동일 색 점 표시

---

## 8) 스마트폰 텍스트 입력 (2모드)

### 8.1 위치 지정형 (정밀 배치)

1. 원하는 위치 터치
2. 텍스트 입력창 활성화
3. 기본 스타일(색/굵기/프리셋) 선택
4. 아래쪽 입력창 입력
5. 전송 누르면 해당 위치에 생성

### 8.2 빠른 텍스트 (연속 작성)

- 우측 하단 텍스트 아이콘 탭 → 위치 클릭 없이 입력 시작
- 조금 전 작성한 텍스트의 아래쪽으로 자동 배치되며 연속 작성됨

---

## 9) 도형/선 + 편집 패턴

### 9.1 도형/선

- 선/화살표/사각형/원 (확장 가능)
- 스타일: 두께/색, 채움/투명도, 점선/실선

### 9.2 편집 (필수)

- 선택/이동/크기 조절(핸들)
- 레이어(앞/뒤)
- 잠금(실수 방지)
- 스냅/격자는 **도형 모드에서만 ON**

---

## 10) 업로드/첨부 (사진/영상/PDF) + 조작

### 10.1 사진 (PNG/JPG)

- 업로드 시 크기 설정
- 롱프레스로 크기 변경/옵션 메뉴
- 투명도 설정

### 10.2 영상

- 크기 조절(사진과 동일)
- 재생 버튼 → 재생, 하단 재생 바

### 10.3 문서 (PDF)

- 크기 조절(사진과 동일)
- 좌/우 반투명 버튼으로 페이지 넘김
- 펼쳐보기(페이지 패널): 아래 또는 우측 확장

### 10.4 오브젝트 롱프레스 공통

- 배경 **Dim 처리**
- 오브젝트 옆 **팝업 메뉴**로 옵션 조정

---

## 11) 시간순 네비게이션 (흐름 파악)

- 화면 내부에 **세로 네비게이션 바**
- 드래그/스크롤 시 해당 시간/구간으로 점프
- (권장) 이벤트 마커: 손글씨/사진/영상/PDF 구분

---

## 12) 모아보기 (태그함)

### 12.1 목록

- 검색
- **필터**: 유형(손글씨/사진/영상), 방, 기간
- **정렬**: 최신순
- **카드**: 썸네일 / 방 이름 / 태그한 사람 / 시간

### 12.2 동작

- 카드 탭 → 해당 방 이동 → 해당 이벤트로 스크롤
- 손글씨는 **태그 영역/시점 하이라이트 + 자동 이동**

---

## 13) 공유/권한/내보내기

### 13.1 링크 공유

- 링크로 페이지(캔버스) 공유
- 앱 미설치: **보기 전용(읽기 전용)**

### 13.2 내보내기

| 항목 | 내용 |
|------|------|
| **포맷** | PDF / PNG / JPG |
| **범위** | 부분 / 전체 |
| **워터마크** | 내보낸 사용자 ID(또는 닉네임) + 앱 이름 |
| **방장 설정** | 내보내기 허용/차단, 워터마크 강제, 로그 공개 범위 |

---

## 14) 실시간/로딩 UX 원칙

- **썸네일/프리뷰 우선** → 원본은 필요 시 로딩
- **전송 상태(권장)**: 전송 중 → 완료 체크 표시
- **네트워크 불안정(권장)**: 재연결 중 배너 + 대기열 표시
