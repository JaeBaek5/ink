# 요청사항 결과 보고서

개발 요청 사항과 진행 결과를 기록하는 폴더입니다.

## 목적

- 개발 진행 내역 추적
- 나중에 어떤 식으로 개발했는지 확인
- 변경 사항 및 결정 사항 문서화

## 보고서 형식

각 보고서는 아래 형식을 따릅니다:

```
# [번호] 요청 제목

## 요청 정보
- 날짜: YYYY-MM-DD
- 요청 내용: ...

## 진행 내용
1. ...
2. ...

## 결과
- ...

## 변경된 파일
- ...

## 관련 커밋
- ...
```
