## 1. 로그아웃 시 예외 및 앱 멈춤 방지

### 요약
- 로그아웃 후 Firestore `permission-denied` 발생 시 예외가 전파되어 앱이 멈추거나 예외 대화상자가 뜨는 현상 방지.
- 멤버 시트(접속 인원 확인) 열 때 RTDB 권한 오류가 나도 앱이 정지하지 않도록 처리.

### 수정 내용
- **firestore_retry.dart**
  - `isPermissionDeniedFirebaseException()` 추가: `permission-denied` / `cloud_firestore/permission-denied` 및 메시지에 "permission" + "denied" 포함 시 true.
  - `streamWithRetry`의 `onError`에서 권한 없음이면 재시도·에러 전파 없이 스트림만 `close()` 하여 예외 대화상자 방지.
- **canvas_screen.dart**
  - 멤버 시트 내 `StreamBuilder<List<String>>`에서 `snapshot.hasError`일 때 빈 목록으로 멤버 목록만 표시(접속 중 표시 생략), 앱 정지 방지.

## 2. 탑바 접속 인원 표시 및 멤버 보기 통일

### 요약
- "N명 접속"을 탑바 제목 옆에 상시 표시하고, 0명이어도 탭 시 멤버 보기 시트가 열리도록 변경.
- 접속 인원 아바타를 탑바에서 가로 스크롤로 표시, 인원이 많을 때 대비.
- 채팅방 설정(더보기) 메뉴에서 "멤버 보기" 항목 제거 → 멤버 보기는 탑바 "N명 접속" 탭으로만 제공.

### 수정 내용
- **canvas_screen.dart**
  - `_buildTitleViewingSection`: `roomViewingIdsStream`이 null이어도 "0명 접속" 표시. 전체를 `GestureDetector`로 감싸 탭 시 `_showMemberListSheet` 호출.
  - 멤버 시트: `showModalBottomSheet` → `showGeneralDialog` + `SlideTransition`으로 **상단에서 슬라이드 다운** 표시. `BorderRadius.vertical(bottom: 16)` 적용.
  - "N명 접속" 텍스트에 `TextDecoration.underline` 추가(탭 가능 힌트).
- **canvas_controller.dart**
  - `_roomViewingIdsStream`을 `_rtdbRoom.roomViewingIdsStream(roomId).asBroadcastStream()`으로 설정하여 앱바·멤버 시트에서 동시 구독 가능 ("Stream has already been listened to" 해결).

## 4. 태그 기능 통일 (손글씨·텍스트·사진)

### 요약
- 손글씨: 길게 누르면 반투명 메뉴 박스 없이 **바로 친구 태그 선택 시트**로 이동.
- 텍스트: 길게 누르면 **태그 추가** + **삭제** 옵션을 가진 바텀시트 표시.
- 사진/영상/PDF: 액션 바 "…" 메뉴를 **팝업 메뉴(showMenu)** 대신 **바텀시트(팝시트)**로 통일하여 선택 흐름 일관화.

### 수정 내용
- **drawing_canvas.dart**
  - 손글씨 롱프레스: `showModalBottomSheet`(반투명 배경 + "@친구 태그" ListTile) 제거 → `_showTagStrokeFriendPicker(context, hit...)` 직접 호출.
  - `_showTextOptions`: ListTile "태그 추가(@친구 태그)" 추가 → 탭 시 `_showTagTextFriendPicker(context, text)` 호출.
  - `_showTagTextFriendPicker`: 텍스트용 `TagModel`(targetType: text, targetId: text.id, areaX/Y from position) 생성 후 태그 저장.
  - `_showMediaMoreMenu`: `showMenu` → `showModalBottomSheet<String>`로 변경. ListTile로 "@친구 태그", "잠금", PDF 시 "한 페이지/여러 페이지" 제공. `Navigator.pop(ctx, value)` 후 기존 분기(tag/lock/pdf_single/pdf_grid) 유지.
